About Smart Resume Matcher

This project focuses on building an AI-powered Resume and Job Matching System that uses embeddings, semantic search, and Generative AI reasoning.

Key Features:

  • Document parsing for PDF, DOCX, TXT files
  • Semantic embeddings using SentenceTransformers
  • Intelligent matching beyond keyword search
  • AI-generated explanations for matches

The system goes beyond traditional keyword matching by using language understanding models to interpret meaning, skills, and experience.
